## PRIME NUMBERS MULTIPLICATION TABLE
```
Problem Set: program that calculates and displays a
multiplication table of the first 10 prime numbers.
```

```
Sample partial output:
2 3 5 7 ...
2 4 6 10 14 ...
3 6 9 15 21 ...
5 10 15 25 35 ...
7 14 21 35 49
```

### How to Run Program

1. git clone the repo
2. cd into the folder cloned
3. run ruby `ruby ./lib/command.rb 'execute'` at the folder root. `i.e prime-table/`
4. input the multiplication table size e.g 1,2,3,4
5. You have your result displayed in a readable and graphical format
6. type `no` to exit the program
